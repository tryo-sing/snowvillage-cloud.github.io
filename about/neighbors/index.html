Neighbors の募集について

Neighborsは、Mayorsと連携しながら、SnowVillageの運営や企画をサポートする「コミュニティのサポーター」です。
それぞれの興味やスキルを活かして、コミュニティ運営、Snowflakeエコシステムの発展のためにぜひお力をお貸しください。

こんな方を求めてます！
✓ Snowflakeが好きで、もっと多くのユーザーとつながりたい方。
✓ コミュニティ運営のノウハウを学びたい、あるいは自分の知見を活かしたい方。
✓ Mayorsをサポートし、メンバー間の活発な交流を生み出したい方。
✓ Data Superheroes、Squad、Mayors を目指して、イベント企画やコンテンツ制作に関わりたい方！

応募にあたっての Q&A

「1年を通じて」とのことですが、仕事や家庭都合など活動レベルが落ちる期間があってもよいですか？

構いません。説明の中で1年としているのは、募集が特定のイベントやポジション別ではないこと、1年の任期で活動いただき、翌年度も続投いただく場合も再度応募〜任命をもって活動していただくことを意図しているためです。

その中で、特定の活動や時期に注力していただくのは問題ありません。

Snowflake 初心者ですが、運営に関わることは可能でしょうか？

初心者の方でも大歓迎です。運営にあたっては、製品技術によらないイベント企画〜開催、サイトメンテナンス、オンライン配信やアーカイブ整備など、さまざまな役割があります。

技術面でも、初心者から取り組んでいる話や、Snowflake に偏らないデータ基盤・IT のノウハウを持った方の目線は大いに期待されています。（技術の話は Neighbors としての活動よりも、ユーザーグループでの活動において個々の目線からの発信が歓迎されています。初心者からの取り組みや、初心者目線でのコンテンツレビューなどもぜひチャレンジしてみてください）

Neighbors に手を挙げたものの、1年間活動できませんでした。

結果的にそうなってしまうことは構いません。次にどんな活動ができるか、どんなことにモチベーションがあるか、どんなスキルを活かせそうかを Mayors と会話して、翌年度も応募するかどうかをご判断ください。

Neighbors になることでどんなメリットがありますか？

ご自身の活動次第です。さまざまなイベントやコンテンツ、運営チーム（ユーザーグループなど）に関わることで、人とつながり、技術知識や視点に触れる機会になります。 Neighbors でなくとも自分次第でこのような経験を積むことはできますが、その機会は大きく広がると考えられます。

公式サイトに Neighbors として掲載しますので、ご自身の評価につなげていただくことも自由です。

Mayors になりたいです。

活動実績、熱意、コミュニティとの関係性は必須条件と考えていますが、明確な規定はありません。（別途公募プロセスを策定予定です。）

まずは Neighbors やユーザーグループとして、やりたいことの実現をお考えください。これまでに SnowVillage での活動実績があり、Mayors として成したいことがあるようでしたら、ぜひ Mayors にご相談ください。

既存の制度やルールを尊重すること、新しいことに取り組み持続可能な制度やルールを作ること、どちらも Mayors に求められる姿勢であり、アクティブな働きかけを期待しています。
